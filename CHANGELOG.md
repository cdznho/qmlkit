# Changelog

All notable changes to qmlkit will be documented in this file.

## 0.1.1 - 2026-04-30

- Implemented usable local `QiskitBackend` execution through Qiskit statevector simulation.
- Implemented usable local `PennyLaneBackend` execution for QNodes and quantum functions.
- Added optional `qiskit`, `pennylane`, and `quantum` dependency extras.

## 0.1.0 - 2026-04-30

- Added `QuantumFeatureMap` and `QuantumKernel` sklearn-compatible feature maps.
- Added differentiable PyTorch `HybridLayer`.
- Added `SimulatorBackend`.
- Added toy benchmark runner, examples, packaging metadata, and tests.
